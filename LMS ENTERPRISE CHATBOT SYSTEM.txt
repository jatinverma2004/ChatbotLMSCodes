LMS ENTERPRISE CHATBOT SYSTEM
Run Instructions (Windows CMD)

1. CREATE / ACTIVATE PYTHON VIRTUAL ENVIRONMENT

cd mcp_v3_server

python -m venv venv

venv\Scripts\activate

2. INSTALL REQUIRED DEPENDENCIES

pip install fastapi uvicorn streamlit requests pypdf python-docx sentence-transformers langgraph


3. START MCP V2 SERVER

cd mcp_v2_server

venv\Scripts\activate

uvicorn app.main:app --reload --port 8000


4. START MCP V3 SERVER

cd mcp_v3_server

venv\Scripts\activate

uvicorn main:app --reload --port 8100


5. START CHATBOT SERVICE


cd chatbot_service

venv\Scripts\activate

uvicorn app:app --reload --port 9000


6. START LANGGRAPH SERVICE (IF USED)

cd chatbot_service

venv\Scripts\activate

uvicorn langgraph_system.langgraph_chat:app --reload --port 9100

7. START CHAT UI

cd chatbot_service

venv\Scripts\activate

streamlit run chat_ui.py

SYSTEM PORTS

MCP V2 SERVER
http://127.0.0.1:8000

MCP V3 SERVER
http://127.0.0.1:8100

CHATBOT API
http://127.0.0.1:9000/chat

LANGGRAPH API
http://127.0.0.1:9100/chat

CHAT UI
http://localhost:8501

IMPORTANT

All servers must run in separate terminals.
Start servers in this order:

1 MCP V3 SERVER
2 CHATBOT SERVICE
3 LANGGRAPH SERVICE
4 STREAMLIT UI
